# AI Planning Documents

This directory contains implementation plans researched and drafted with AI assistance.
Each plan captures research findings, architectural decisions, and phased implementation steps.

| Document | Status | Summary |
|----------|--------|---------|
| [001-incoming-pipeline.md](001-incoming-pipeline.md) | Planning | AMQP filter pipeline for incoming ActivityPub activities |
| [002-delivery-feedback.md](002-delivery-feedback.md) | Planning | publisherd delivery reports and error feedback system |
| [003-timelines.md](003-timelines.md) | Planning | Home/local/federated timelines and collection pagination |
| [004-pkgfed.md](004-pkgfed.md) | Planning | Subscription-based package update notifications via ActivityPub |

## Conventions

- Plans are numbered sequentially: `NNN-short-name.md`
- Status values: `Planning`, `Approved`, `In Progress`, `Done`
- Each plan has a research section (findings) and an implementation section (phases)
- Update this README when adding or completing plans
